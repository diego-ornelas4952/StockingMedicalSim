# StockingMedicalSim
Stocking system for the Simulators Laboratory in CUAltos (it's for my social service)

# How it works?
It's really simple. It's only an admin user, and a student role.
    - The admin can add items to the stock and register more users
    - The student only can check the stock in the list, adding comments, and download the PDF with the report

## Tech Stack
    - React
    - Vite
    - TailwindCSS

## Run
    - npm run dev
